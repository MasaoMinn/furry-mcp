# Agent Skills

These documents are shipped with the npm package so users can install the MCP server and get the companion state instructions in the same download.

Files:

- `claude.md`
- `codex.md`
- `cursor.md`

Important: MCP clients do not automatically activate arbitrary skill files from npm packages. The files are included for distribution and can be copied into the agent-specific skill or instruction location used by Claude, Codex, or Cursor.
